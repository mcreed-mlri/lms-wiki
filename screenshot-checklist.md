## Tips for Taking Screenshots

- **Use Student Preview** (or a test account) to capture the learner-facing views — these look different from the editor views.
- **Window width:** Take screenshots at a consistent browser width. About 1280px wide produces readable text without huge empty margins.
- **Zoom in** on the relevant area if the full screen is too cluttered — a focused crop is more useful than a screenshot of the entire screen.
- **Annotation tool:** Use a tool like Snagit, Greenshot, or the Windows Snipping Tool to add arrows or circles highlighting the key element before saving.
- **Naming:** Save files exactly as named above so you can match them to the guide when you're ready to insert them.
