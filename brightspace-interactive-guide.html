Brightspace Native Tools Reference

Optional Brightspace-native features to consider when custom HTML is not the best fit.

Executive Summary

LACE's default course model is custom HTML. Brightspace hosts, enrolls, tracks, and reports. The tools below are optional Brightspace-native options, not the primary course-authoring model.

Use this page when deciding whether a native Brightspace feature is worth using for a specific activity: quizzes, surveys, discussions, assignments, checklists, email automation, or simple H5P interactions.

NeedDefaultConsider Native Brightspace When
Course pages and activitiesCustom HTMLThe interaction must be graded or tracked by Brightspace directly.
Knowledge checksCustom HTML or lightweight quizYou need LMS-gradebook scoring, attempts, or item-level reporting.
Reflection / feedbackCustom HTML or survey formYou need responses stored in Brightspace.
DiscussionCustom HTML prompt + team facilitationYou need Brightspace-hosted threaded discussion.
Completion recordsBrightspace completion trackingAlways use the LMS record as the source of truth.

What Is Brightspace?

Brightspace (made by D2L) is the platform where you build online and blended training modules. Learners enrolled in your trainings can watch videos, complete activities, read materials, and interact with each other — all in one place.

What Is Creator+?

Creator+ is an upgrade that gives your organization access to H5P — a library of interactive content tools. With H5P you can embed questions inside videos, create drag-and-drop exercises, build flashcard decks, and more, all without any coding.

How Training Is Organized in Brightspace

For LACE's operating structure, use the Team Wiki hierarchy: Training Area, Program, Course Template, Course Offering, optional Section.

  • Training Modules — the top-level container for a topic (e.g., "Eviction Defense Fundamentals")
  • Units / Sections — chapters or topic groupings within the module
  • Content Items — the individual pieces: videos, readings, activities, and interactive exercises

See the Course Content Model for the source of truth.

Throughout this guide, participants are referred to as learners and outcomes as training completions.
Manage Files vs. Course Builder: deleting a file from Manage Files does not automatically remove the course outline item that pointed to it. If a page is retired, remove the file and the Course Builder/Content node. If the page is still used, overwrite the file instead of deleting it.

Navigating to the Content Editor

Before you can add any interactive elements, you need to open your training module and reach the content editor. All of the tools in this guide are accessed from there.

Opening a Training Module

  1. Log in to Brightspace at your organization's URL.
  2. From your dashboard, click the name of the training module you want to edit.
  3. Click the Content tab in the top navigation of the module.

H5P Interactive Content Creator+

H5P is a library of ready-made interactive activities you can embed directly into your training content. With Creator+, your organization has access to H5P at no additional cost.

H5P activities are well-suited for legal skills training because they:

  • Let learners engage with material actively, not just read or watch passively
  • Allow practice and self-checking without any grade pressure
  • Can be retaken as many times as an learner wants
  • Work on phones, tablets, and computers
  • Are built once and reusable across multiple training modules

The following H5P activity types are covered in this guide:

H5P Type What Learners Do Great For
Interactive Video Answer questions embedded at points in a video Video walkthroughs of procedures
Image Hotspots Click labeled spots on an image to reveal information Annotating forms, diagrams, documents
Flashcards Flip through digital cards to review terms Legal vocabulary, key concepts
Drag and Drop Drag items to correct zones on a background Matching, sorting, categorizing exercises
Course Presentation Click through slides with embedded activities Structured mini-modules
Accordion Expand and collapse content sections Reference material, FAQs, procedures

How to Insert an H5P Activity

All H5P activities are added to content items using the same starting steps. The steps below get you to the H5P activity library, where you choose which type to create.

  1. Open a content item in the Content Editor (see previous section).
  2. Click Insert in the editor toolbar.
  3. Select Insert Stuff from the dropdown menu.
  4. In the window that opens, look for H5P in the list of options and click it.
  5. Click Create New to build a new activity, or Add Existing to reuse one you've already made.
  6. Choose the H5P activity type you want from the list (each type is covered in the sections below).

H5P: Interactive Video Creator+

What It Is

A video with questions, notes, or clickable markers embedded at specific moments in the timeline. When the video reaches that point, it pauses and the learner must respond before it continues.

When to Use It

Any time you have a training video and want to make sure learners are actively following along — for example, a video walkthrough of a client intake procedure with comprehension checks at key points.

How to Build One

  1. Follow the H5P insertion steps above and select Interactive Video from the activity list.
  2. Add your video by uploading a file, or by pasting a video URL (YouTube, Vimeo, or a direct video link).
  3. Click the + button on the video timeline to add an interaction at any moment in the video.
  4. Choose the type of interaction:
    • Multiple Choice Question — pause the video and present a question
    • Fill in the Blank — learners complete a sentence or key phrase
    • Text Overlay — display a note, key point, or definition
    • Bookmark — add a labeled navigation marker (like a chapter marker)
  5. Fill in your question text and answer choices. Mark the correct answer if applicable.
  6. Optionally add feedback text for correct and incorrect answers.
  7. Click Save when finished.

Keep embedded questions short and focused — one key concept per pause point. Two or three questions spread across a 10-minute video is usually the right balance. More than that can feel like an exam rather than a training.

H5P: Image Hotspots Creator+

What It Is

An image with clickable marker spots that reveal information when an learner clicks them. Each hotspot can display a short explanation, a definition, a link, or an image.

When to Use It

Annotating documents, forms, or diagrams — for example, a sample client intake form where each section has a hotspot explaining what information goes there and why it matters, or a courtroom layout explaining where each party stands.

How to Build One

  1. Follow the H5P insertion steps and select Image Hotspots from the activity list.
  2. Upload your background image (e.g., a sample form, a workflow diagram, or a document page).
  3. Click anywhere on the image to place a hotspot marker at that location.
  4. For each hotspot, fill in:
    • Header — a short label (e.g., "Client Name Field")
    • Content — the explanation text that appears when clicked
  5. Optionally change the hotspot icon style (circle, plus, etc.) and color.
  6. Repeat for each spot you want to annotate on the image.
  7. Click Save when finished.

H5P: Flashcards Creator+

What It Is

A digital flashcard deck. Each card shows a term, question, or prompt on the front. The learner clicks to flip the card and see the definition, answer, or explanation on the back.

When to Use It

Building vocabulary (legal terms, Latin phrases, statute names), reviewing procedural steps, or any situation where quick recall is useful. A "Client Rights" deck or "Common Intake Forms" deck can be referenced any time during the training program.

How to Build One

  1. Follow the H5P insertion steps and select Flashcards from the activity list.
  2. Enter a title for the deck (e.g., "Housing Law — Key Terms").
  3. Click Add Card to create the first card.
  4. For each card, enter:
    • Front text — the term, question, or prompt learners see first
    • Back text — the definition, answer, or full explanation
    • Optionally upload an image to either side of the card
  5. Click Add Card again to continue adding cards to the deck.
  6. Click Save when the deck is complete.

H5P: Drag and Drop Creator+

What It Is

An activity where learners drag labeled items and drop them into the correct zones on a background image or grid. Immediate feedback is shown when the activity is submitted.

When to Use It

Matching and sorting exercises — for example: matching the steps of an eviction proceeding to the correct timeline stage, sorting different types of evidence into admissible vs. inadmissible categories, or matching legal terms to their definitions.

How to Build One

  1. Follow the H5P insertion steps and select Drag and Drop from the activity list.
  2. Upload a background image, or choose a blank grid layout.
  3. Define Drop Zones — click and drag on the background to draw the rectangular areas where items can be dropped. Label each zone.
  4. Add Draggable Items — these are the items learners will drag. Each item can be a text label or an image.
  5. Connect each draggable item to its correct drop zone (this tells Brightspace what the right answer is).
  6. Set feedback messages for correct and incorrect responses (optional but helpful).
  7. Click Save when finished.

H5P: Course Presentation Creator+

What It Is

A slide-based presentation where each slide can contain text, images, videos, and embedded interactive H5P elements. Learners click through the slides at their own pace.

When to Use It

Building a structured mini-module within a training — for example, a 10-slide overview of the eviction defense process with comprehension checks embedded in specific slides. Course Presentations let you combine explanatory content and practice in a single, self-contained activity.

How to Build One

  1. Follow the H5P insertion steps and select Course Presentation from the activity list.
  2. Use the slide panel on the left to add, reorder, and delete slides.
  3. For each slide, use the toolbar to add text, images, video, and links.
  4. To embed an interactive element on a slide, click the + button and choose an H5P type (e.g., Multiple Choice, Drag and Drop).
  5. In the Navigation settings, choose whether learners can skip ahead freely or must complete each slide before moving on.
  6. Add a summary slide at the end with key takeaways from the presentation.
  7. Click Save when finished.

H5P: Accordion Creator+

What It Is

A list of expandable content sections. Each item has a clickable header — learners click it to expand and read the content beneath. Collapsed sections stay hidden until clicked, keeping the page clean and scannable.

When to Use It

Reference material, FAQs, step-by-step procedures for complex tasks, or any content where you want to organize a lot of information without overwhelming the reader at once. An Accordion is excellent for a "Procedures Reference" panel that learners can revisit throughout training.

How to Build One

  1. Follow the H5P insertion steps and select Accordion from the activity list.
  2. Click Add Item to create your first section.
  3. For each item, fill in:
    • Title — the clickable header learners see (keep it short and descriptive)
    • Content — the text that appears when the section is expanded; can include bullet lists, links, and images
  4. Repeat for each section you want to include.
  5. Click Save when finished.

Quizzes — Knowledge Checks

In Brightspace this tool is called Quizzes, but for legal training purposes think of them as knowledge checks — low-pressure activities that help learners confirm their understanding. They do not need to count toward any score.

Knowledge checks work well after a video or reading to help learners confirm they absorbed the key points, before a training topic to gauge starting knowledge, or at the end of a module as a self-assessment.

Creating a Knowledge Check

  1. Click Tools in the top navigation of your training module.
  2. Select Quizzes from the dropdown.
  3. Click New Quiz.
  4. Give the quiz a descriptive name (e.g., "Intake Procedure — Knowledge Check").
  5. Click Add/Edit Questions to begin adding questions.
📷
Screenshot: The Quizzes tool in the module navigation, with "New Quiz" button visible

Question Types

Question Type Best For
Multiple Choice Testing knowledge of a specific fact or concept; one correct answer
True / False Quick checks on common misconceptions
Multi-Select Questions where more than one answer can be correct
Short Answer Open-ended reflection prompts; learner types a free response
Matching Connecting terms to definitions, or steps to outcomes
Ordering Arranging steps or events in the correct sequence

Adding a Question

  1. Click New → New Question → choose your question type.
  2. Type the question in the Question Text field.
  3. Add answer options. For multiple choice, check the box next to the correct answer.
  4. Optionally add feedback text for correct and incorrect answers — this is shown to learners immediately after they respond and is a valuable teaching moment.
  5. Click Save.
📷
Screenshot: The question editor for a multiple choice question — question text, answer options, and correct-answer checkbox
📷
Screenshot: The feedback fields for correct and incorrect answers

Setting the Quiz as Ungraded

  1. In the quiz settings, click the Assessment tab.
  2. Under Grade Item, select [None] — this removes the quiz from the gradebook entirely.
  3. Alternatively, set the quiz to automatically award full credit on submission so learners aren't penalized for their answers.
📷
Screenshot: The Assessment tab with Grade Item set to [None]

Allowing Unlimited Retakes

  1. In the quiz settings, click the Attempts tab.
  2. Set Attempts Allowed to Unlimited so learners can retake the check as many times as they want.
  3. Click Save and Close.
📷
Screenshot: The Attempts tab with "Unlimited" selected
Even when a quiz is ungraded, Brightspace still records completion. You can see which learners finished the knowledge check without assigning any score — useful for tracking engagement and identifying who may need follow-up.

Surveys — Feedback and Reflection Tools

Surveys let you collect input from learners in a way that is separate from the quiz tool. Responses are typically anonymous or confidential, making surveys better for honest feedback.

Survey vs. Quiz — What's the Difference?

  • Quizzes have right and wrong answers. Surveys do not.
  • Surveys are better suited for gathering opinions, training evaluations, pre-training needs assessments, and open reflection prompts.
  • Survey results can be kept anonymous so individual learners' responses stay private.

Creating a Survey

  1. Click Tools → Surveys.
  2. Click New Survey.
  3. Enter a name (e.g., "Training Feedback — Eviction Defense Module").
  4. Click Add/Edit Questions to start adding questions.
📷
Screenshot: The Surveys tool with the "New Survey" button visible

Useful Survey Question Types

Question Type Example Use
Likert / Rating Scale "How confident do you feel about advising a client on [topic]?" (rate 1–5)
Short Answer "What was the most useful part of this training?"
Multiple Choice "Which topic would you like more training on?" (choose one)
Multi-Select "Which formats work best for your learning?" (select all that apply)
📷
Screenshot: The survey question editor showing a Likert / Rating Scale question being configured

Enabling Anonymous Responses

  1. In the survey settings, find the Responses section.
  2. Check Allow anonymous responses.
  3. With this enabled, you can view aggregated results but not individual learners' answers.
📷
Screenshot: The survey settings panel with the "Allow anonymous responses" checkbox
A short 3-to-5 question survey at the end of each training module takes learners less than two minutes and gives you data to continuously improve your trainings. Consider a "confidence before and after" question on a 1–5 scale — it's a simple way to demonstrate training impact over time.

Discussions

Discussions give learners a space to post responses, ask questions, and learn from each other's approaches. Discussions in Brightspace are organized into Forums (broad topic containers) and Topics (the specific threads where learners post).

Forum vs. Topic

  • A Forum is like a folder — it groups related discussion topics together.
    Example forum: "Case Scenario Practice"
  • A Topic is the actual discussion thread where learners write their posts.
    Example topic: "Module 2 — How would you handle this client interview situation?"

Creating a Forum

  1. Click Tools → Discussions.
  2. Click New → New Forum.
  3. Enter a forum name and a brief description so learners understand its purpose.
  4. Click Save and Add Topic to immediately create the first topic within this forum.
📷
Screenshot: The Discussions tool with the New Forum option in the dropdown
📷
Screenshot: The New Forum form with name and description fields

Creating a Discussion Topic

  1. Inside a forum, click New Topic.
  2. Enter the topic title (short) and a description / prompt — this is the text learners see before they post. Make the prompt specific and scenario-based.
  3. Under Options, consider enabling:
    • "Require a post before reading replies" — learners must submit their own response before seeing what others wrote; this encourages original thinking
    • "Allow anonymous posts" — creates a safer space for sensitive questions
  4. Click Save and Close.
📷
Screenshot: The New Topic form — title, prompt/description, and the Options panel
📷
Screenshot: The "Require a post before reading replies" option checked in Topic Options

Linking a Discussion to Your Training Content

  1. Open your training module's content area.
  2. Click Add Existing Activities.
  3. Select Discussions and choose the topic you created.
  4. This adds a direct link to the discussion thread from within the training flow, so learners don't need to hunt for it.
📷
Screenshot: The "Add Existing Activities" menu with Discussions highlighted

Facilitation Tips

  • Post a short opening message in the thread to model the tone and depth you're looking for.
  • Set up email notifications so you're alerted when someone posts — this lets you respond quickly, which signals to learners that engagement is valued.
  • Respond to at least a few posts publicly to validate contributions and add nuance.
  • Use the Moderate Posts setting if you want to review posts before they're visible to others.
A scenario-based discussion prompt — "Read this client intake situation and post what you would do first, and why" — is one of the most effective activities for legal skills training. It surfaces different approaches, creates peer learning, and gives you insight into where learners need more support.

Assignments and Reflections

In Brightspace, the Assignments tool (also called the Assignment Dropbox) is where learners submit work. For legal training purposes, "work" usually means a reflection, a written response to a scenario, or a practice document — not homework in the traditional sense.

Creating an Assignment

  1. Click Tools → Assignments.
  2. Click New Assignment.
  3. Enter an assignment name and write clear instructions in the description field. Tell learners exactly what you're asking them to submit and how long it should be.
  4. Choose a submission type (see table below).
  5. Click Save and Close.
📷

Submission Types

Submission Type When to Use It
File Submission Learners upload a Word document, PDF, or other file — good for draft memos, sample client letters, or practice pleadings
Text Submission Learners type directly in the browser — good for short reflections or written responses
Video/Audio Recording Learners record themselves — good for oral skills practice, such as a mock client explanation
On Paper / Observed in Person Record that a live skill demonstration was observed — useful for in-person components of a blended training

Setting the Assignment as Ungraded

  1. In the assignment settings, find the Assessment section.
  2. Set the Grade Item to [None].
  3. Learners can still submit, and you can still read their submissions — there's simply no score attached.

Adding a Rubric (Optional)

Even without a grade, a rubric helps learners understand what a quality reflection or submission looks like — it becomes a self-assessment checklist rather than a grading tool.

  1. In the assignment editor, scroll to Evaluation and Feedback.
  2. Click Add Rubric.
  3. Select an existing rubric or click Create Rubric to build one.
📷
Screenshot: The "Add Rubric" option in the Evaluation and Feedback section
A simple completion reflection — two to three sentences about one thing they learned and one thing they'll do differently — is low-barrier and creates a meaningful record of training engagement. It also helps learners consolidate their learning at the end of a module.

Checklists

A Checklist in Brightspace is a to-do list that learners can check off as they complete training items. It helps them track their own progress and gives them a clear view of what's required.

Checklists work well for multi-part training programs, new staff onboarding, and any compliance or certification training where completion tracking matters.

Creating a Checklist

  1. Click Tools → Checklists.
  2. Click New Checklist.
  3. Enter a checklist name (e.g., "New Staff Onboarding — Training Checklist").
  4. Optionally add a description explaining how to use the checklist.
📷
Screenshot: The Checklists tool with the "New Checklist" button

Adding Categories and Items

Checklists are organized by optional Categories (section headers) and Items (individual tasks).

  1. Click Add Category to create a section header (e.g., "Week 1 — Core Modules," "Required Readings").
  2. Click Add Item under a category.
  3. For each item, enter:
    • The item name (e.g., "Watch: Client Interview Overview Video")
    • An optional due date
    • An optional description or link for more context
  4. Repeat for all items across all categories.
📷
Screenshot: The Checklist editor with two categories and several items listed under each
📷
Screenshot: The Add Item form with name, due date, and description fields

Linking Items to Content (Auto-Completion)

You can connect a checklist item directly to a Brightspace content item so it checks off automatically when the learner completes that activity.

  1. When adding or editing a checklist item, look for the Linked Object or Completion option.
  2. Select the corresponding content item, quiz, or activity.
  3. When the learner completes that item in the module, the checklist updates automatically — no manual checking required.
📷
Screenshot: The Linked Object option in the checklist item settings — connecting the item to a content activity

How Learners Use the Checklist

  1. Link the checklist from your training module's content area using Add Existing Activities → Checklists.
  2. Learners see the checklist with empty checkboxes they can mark off manually, or items that check automatically if linked.
  3. Progress is visible to both learners and trainers.
📷
Screenshot: How the checklist appears to an learner — checkboxes next to each item, some checked
Pin the checklist at the top of your training module's content area. Many learners find it helpful to see the full scope of a training before they start, so they can plan when to complete each part. It also reduces the anxiety of not knowing "how much is left."
