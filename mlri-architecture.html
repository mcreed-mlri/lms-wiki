  • Manager team views
  • Program completion rates
  • Cohort equity analysis
  • Content effectiveness
  • Catalog gap analysis
LACE Hub — one app, role-based views

Dashboards are built into LACE Hub, not as separate Brightspace, Supabase, or Monday products. A single API layer merges Brightspace + Supabase; each user sees only their role's screens.
